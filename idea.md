# Patient Health Risk Assessment System

## 1. Project Overview

The Patient Health Risk Assessment System is a full-stack healthcare application that enables healthcare staff to assess patient health risks using predictive analysis and generate structured health reports with personalized recommendations.

The system focuses primarily on backend design, following strong software engineering principles including OOP, layered architecture, and clean separation of concerns.

---

## 2. Problem Statement

Healthcare institutions require efficient systems to:
- Store patient clinical data
- Perform risk assessment
- Generate reports
- Provide evidence-based recommendations

Manual processing is error-prone and inefficient.

This system aims to automate the risk assessment workflow.

---

## 3. Target Users

- Healthcare Staff (Doctors / Admin)
- Patients
- External Medical Knowledge Systems

---

## 4. Core Features

### Data Management
- Add patient information
- Upload clinical data

### Risk Processing
- Predict health risk
- Generate health report
- Generate personalized recommendations
- Retrieve medical knowledge from external system

### Patient Access
- View risk prediction
- View health report
